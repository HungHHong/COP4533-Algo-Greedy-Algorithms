## Overview
This project implements and compares **three cache eviction algorithms** used in paging systems. The goal is to simulate how different strategies manage a cache with limited capacity and measure the number of cache misses generated by each algorithm.

The implemented algorithms include:
1.  **FIFO (First-In, First-Out):** Evicts the page that has been in the cache the longest.
2.  **LRU (Least Recently Used):** Evicts the page that has not been used for the longest time.
3.  **OPTFF (Optimal Farthest-in-Future / Belady’s Algorithm):** Evicts the page whose next use occurs farthest in the future.

    -*This algorithm is optimal but requires knowledge of future requests.*

**The program processes a sequence of page requests and reports the number of cache misses for each eviction strategy.**
